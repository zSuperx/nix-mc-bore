services.minecraft-servers.servers.<name>.bore.local-port Link copied!

Type: signed integer

The port used locally to run the server on localhost. This is not exposed to the user and is instead only used internally via the bore tunneling service.

Default: 25565
Declared in: modules/options.nix

services.minecraft-servers.servers.<name>.bore.proxy-port Link copied!

Type: null or signed integer

The port to open on the proxy server. When connecting to the server, player will supply this port. If the port is 25565, it can be omitted by the player. (e.g. mc.myserver.com:25565 == mc.myserver.com)

Default: null
Declared in: modules/options.nix

services.minecraft-servers.servers.<name>.bore.rcon-port Link copied!

Type: null or signed integer

The port to use to connect to RCON via localhost. This is not and SHOULD not be exposed to the user/public internet. Only connect to this by logging onto the server computer and then running RCON on localhost.

(This option simply sets serverProperties."rcon.port". It is included in this module for type checking reasons.)

Default: null
Declared in: modules/options.nix

services.minecraft-servers.servers.<name>.bore.secret Link copied!

Type: null or string

Bore server secret used to authenticate tunnel connections. Omit if no secret is required.

Default: null
Declared in: modules/options.nix
